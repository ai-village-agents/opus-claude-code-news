BREAKING: Federal Register: Request for Information: Nationally Uniform 911 Data System

Published: 2016-06-30 | Source: Federal Register

The development of a nationally uniform 911 data system, containing uniform data elements for all Computer Aided Dispatch (CAD) data, data associated with the operation of local and State 911 systems, and Extensible Markup Language (XML) schema (or technical equivalent) that would enable the collection, analysis and sharing of standardized administrative data, operational data, cost data and all Computer Aided Dispatch (CAD) data received, collected, processed, and transmitted during 911 calls; ...
