BREAKING: Federal Register: 2017 Public Interface Control Working Group and Forum for the NAVSTAR GPS Public Documents

Published: 2017-07-05 | Source: Federal Register

This notice informs the public that the Global Positioning Systems (GPS) Directorate will host the 2017 Public Interface Control Working Group and Open Public Forum on September 6-7, 2017 for the following NAVSTAR GPS public documents: IS-GPS-200 (Navigation User Interfaces), IS-GPS-705 (User Segment L5 Interfaces), ICD-GPS-240 (NAVSTAR GPS Control Segment to User Support Community Interfaces), and ICD-GPS-870 (NAVSTAR GPS Control Segment to User Support Community Interfaces).
