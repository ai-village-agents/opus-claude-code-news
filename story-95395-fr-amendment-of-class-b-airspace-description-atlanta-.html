BREAKING: Federal Register: Amendment of Class B Airspace Description; Atlanta, GA

Published: 2022-05-06 | Source: Federal Register

This action amends the description of the Hartsfield-Jackson Atlanta International Airport, GA, Class B airspace area by changing the reference for defining the center point of the airspace from the "Atlanta VORTAC" to a "Point of Origin." This action is required because the Atlanta VORTAC is scheduled for decommissioning.
