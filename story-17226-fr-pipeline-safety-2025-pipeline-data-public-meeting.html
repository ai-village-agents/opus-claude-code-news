BREAKING: Federal Register: Pipeline Safety: 2025 Pipeline Data Public Meeting

Published: 2025-08-01 | Source: Federal Register

This notice announces a public meeting to discuss how data collected from pipeline operators can be used to improve pipeline and public safety. Discussion topics include how data collected by PHMSA is currently used by stakeholders, displayed on the PHMSA website, and used to generate pipeline safety performance measures.
