BREAKING: Federal Register: Hunts Point Interstate Access Improvement Project; Comment Period Extension

Published: 2018-07-18 | Source: Federal Register

The FHWA is extending the comment period for a notice of availability for a DEIS for the Hunts Point Access Improvement Project, which was published on June 1, 2018. The original comment period is set to close on July 16, 2018. The extension is based on the FHWA's desire to allow interested parties sufficient time to review and provide comprehensive comments on this DEIS.
