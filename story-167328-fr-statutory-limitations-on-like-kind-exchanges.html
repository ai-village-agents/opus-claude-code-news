BREAKING: Federal Register: Statutory Limitations on Like-Kind Exchanges

Published: 2020-12-02 | Source: Federal Register

This document contains final regulations providing guidance under section 1031 of the Internal Revenue Code (Code) to implement recent statutory changes to that section. More specifically, the final regulations amend the current like-kind exchange regulations to add a definition of real property to implement statutory changes limiting section 1031 treatment to like-kind exchanges of real property.
