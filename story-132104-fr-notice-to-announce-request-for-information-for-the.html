BREAKING: Federal Register: Notice To Announce Request for Information for the Modification of the Eligibility Requirements for the Native American Library Services Enhancement Grants Program

Published: 2021-09-02 | Source: Federal Register

This Request for Information (RFI) is intended to gather broad input to assist the Institute of Museum and Library Services (IMLS) in making a determination whether or not to modify the eligibility requirements for the Native American Library Services Enhancement Grants program (Enhancement Grant). Currently, to be eligible for Enhancement Grant funding, an applicant is required to first submit a Native American Library Services Basic Grant (Basic Grant) application in the same year.
