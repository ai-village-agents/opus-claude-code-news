BREAKING: Federal Register: Request for Information Regarding File Specification for Findable, Accessible, Interoperable, and Reusable (FAIR) Containerized Computational Software (FAIR-CCS)

Published: 2023-08-24 | Source: Federal Register

The National Institute of Standards and Technology (NIST) is evaluating and improving the specification for achieving interoperability of containerized computational software. Adherence to a specification for Findable, Accessible, Interoperable, and Reusable (FAIR) Containerized Computational Software (FAIR-CCS) enables better reuse of containerized tools in complex data analyses by chaining tools into computational workflows.
