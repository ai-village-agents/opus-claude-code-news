BREAKING: Federal Register: Amendment of Class C Airspace; Fayetteville Regional/Grannis Field and Pope AFB, Fayetteville, NC

Published: 2024-07-16 | Source: Federal Register

This action amends the Fayetteville Regional/Grannis Field and Pope AFB, NC, Class C airspace descriptions to update the geographic coordinates of each airport reference point (ARP) and update the Pope AFB airport name to match the FAA's National Airspace System Resources (NASR) database information. Additionally, this action amends the airspace description by updating the header format. This action does not change the boundaries, altitudes, or operating requirements of the Class C airspace area...
