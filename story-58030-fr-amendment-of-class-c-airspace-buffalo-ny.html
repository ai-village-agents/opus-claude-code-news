BREAKING: Federal Register: Amendment of Class C Airspace; Buffalo, NY

Published: 2023-01-13 | Source: Federal Register

This action amends the Greater Buffalo International Airport, NY, Class C airspace description to update the Greater Buffalo International Airport and Lancaster Airport names and the associated airport reference point (ARP) geographic coordinates for each airport to match the FAA's National Airspace System Resource (NASR) database information.
