BREAKING: Federal Register: Implementation of Changes to the Parole Process for Venezuelans

Published: 2023-01-09 | Source: Federal Register

This notice announces that the Secretary of Homeland Security (Secretary) has authorized updates to the Parole Process for Venezuelans that was initiated in October 2022.
