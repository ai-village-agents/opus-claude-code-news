BREAKING: Federal Register: Conservation Stewardship Program (CSP) Interim Rule

Published: 2014-12-23 | Source: Federal Register

NRCS and CCC published an interim final rule amending the existing regulation for the Conservation Stewardship Program with request for comment, with a comment period ending January 5, 2015. This document extends the comment period.
