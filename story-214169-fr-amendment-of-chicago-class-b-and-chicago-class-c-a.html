BREAKING: Federal Register: Amendment of Chicago Class B and Chicago Class C Airspace; Chicago, IL

Published: 2018-08-16 | Source: Federal Register

This action amends the Chicago, IL, Class B airspace area description by changing references for defining the center point of the Chicago Class B airspace from the Chicago O'Hare VHF Omnidirectional Range/Distance Measuring Equipment (VOR/DME) to "Point of Origin." The "Point of Origin" uses the same geographic coordinates as the Chicago O'Hare VOR/DME location.
