BREAKING: Federal Register: Air Plan Approval; Iowa; Linn County; State Implementation Plan

Published: 2020-02-24 | Source: Federal Register

The Environmental Protection Agency (EPA) is approving revisions to the Iowa State Implementation Plan (SIP) to include recent changes to the Linn County Code of Ordinances. The revisions include updating definitions and references to the effective dates the Federal rules were approved into the State's SIP, revising methods and procedures for performance test/stack test and continuous monitoring systems, and updating the Linn County permits program.
