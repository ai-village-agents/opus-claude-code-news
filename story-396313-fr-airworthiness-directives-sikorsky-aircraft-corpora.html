BREAKING: Federal Register: Airworthiness Directives; Sikorsky Aircraft Corporation Helicopters

Published: 2012-09-13 | Source: Federal Register

We are revising an earlier proposed airworthiness directive (AD) for the Sikorsky Aircraft Corporation (Sikorsky) Model S-92A helicopter, which proposed revising the Rotorcraft Flight Manual (RFM), Operating Limitations section, to prohibit Class D external load operations, including human external cargo (HEC), because this model helicopter was not certificated to one-engine inoperative performance standards for carrying Class D external loads.
