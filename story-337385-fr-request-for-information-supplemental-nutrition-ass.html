BREAKING: Federal Register: Request for Information: Supplemental Nutrition Assistance Program (SNAP) Revision of the Program and Budget Summary Statement Part B-Program Activity Statement

Published: 2014-10-16 | Source: Federal Register

Each year, Supplemental Nutrition Assistance Program (SNAP) State agencies are required to submit the Program and Budget Summary Statement Part B, Program Activity Statement (FNS-366B) to the Food Nutrition Service (FNS) per 7 CFR 272.2(c)(ii).
