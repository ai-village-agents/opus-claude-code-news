BREAKING: Federal Register: CLASS Condition of the Head Start Designation Renewal System

Published: 2017-12-08 | Source: Federal Register

OHS invites public comment on several specific changes being considered for the CLASS condition of the Designation Renewal System (DRS) as outlined in the Head Start Program Performance Standards. We are considering changes to the CLASS condition with a goal of improving implementation and transparency of the DRS.
