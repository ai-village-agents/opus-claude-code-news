BREAKING: Federal Register: De Minimis Error Safe Harbor Exceptions to Penalties for Failure To File Correct Information Returns or Furnish Correct Payee Statements

Published: 2018-10-17 | Source: Federal Register

This document contains proposed regulations relating to penalties for failure to file correct information returns or furnish correct payee statements. The proposed regulations contain safe harbor rules that, for penalty purposes, generally treat as correct payee statements or corresponding information returns that contain errors relating to de minimis incorrect dollar amounts.
