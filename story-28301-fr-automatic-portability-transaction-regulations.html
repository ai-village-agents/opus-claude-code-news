BREAKING: Federal Register: Automatic Portability Transaction Regulations

Published: 2024-01-29 | Source: Federal Register

This document contains a proposed rule that would implement the statutory prohibited transaction exemption under section 4975 of the Internal Revenue Code (Code) for certain automatic portability transactions. Section 120 of the SECURE 2.0 Act of 2022 amended Code section 4975 to add a statutory exemption for the receipt of fees and compensation by an automatic portability provider for services provided in connection with an automatic portability transaction.
