BREAKING: Federal Register: Homeland Security Academic Partnership Council

Published: 2023-04-05 | Source: Federal Register

The Secretary of Homeland Security (Secretary) approved the Homeland Security Academic Advisory Council (HSAAC) name change to Homeland Security Academic Partnership Council (HSAPC, hereinafter "Council") to avoid confusion with the name of the Homeland Security Advisory Council (HSAC).
