BREAKING: Federal Register: Steel Import Monitoring and Analysis System

Published: 2016-10-13 | Source: Federal Register

The Department of Commerce (the Department) publishes this proposed rule to request public comments on proposed modifications to the regulations for the Steel Import Monitoring and Analysis (SIMA) System that would extend the system until March, 2022.
