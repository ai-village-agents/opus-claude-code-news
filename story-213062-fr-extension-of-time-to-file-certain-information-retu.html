BREAKING: Federal Register: Extension of Time To File Certain Information Returns

Published: 2018-08-03 | Source: Federal Register

This document contains final regulations providing rules regarding the automatic and non-automatic extension of time to file certain information returns. These changes are being implemented to accelerate the filing of the Form W-2 series (except Form W-2G) and forms that report nonemployee compensation (currently Form 1099-MISC with information in box 7) so they are available earlier in the filing season for use in the IRS's identity theft and refund fraud detection processes.
