BREAKING: Federal Register: Request for Comment; Draft Model Minimum Uniform Crash Criteria (MMUCC) Guideline, Sixth Edition

Published: 2023-02-02 | Source: Federal Register

The Model Minimum Uniform Crash Criteria (MMUCC) provides States a guideline for describing crashes involving motor vehicles in- transport to generate the information necessary to improve traffic safety. The primary benefit of using MMUCC is increased crash data uniformity so traffic safety insights can be developed more quickly. Since its inception in 1998, MMUCC has been a voluntary guideline for States; however, standardization of crash data is essential to NHTSA and its safety stakeholders. ...
