BREAKING: Federal Register: Amendment of Class C Airspace; Evansville, IN

Published: 2022-11-22 | Source: Federal Register

This action amends the Evansville Regional Airport, IN, Class C airspace description to update the airport reference point (ARP) geographic coordinates for the Evansville Regional Airport and the Skylane Airport to match the FAA's National Airspace System Resource (NASR) database information.
