BREAKING: Federal Register: Security Considerations for Lavatory Oxygen Systems

Published: 2012-02-27 | Source: Federal Register

On March 8, 2011, the FAA published an interim final rule, request for comments (Amendment Nos. 21-94, 25-133, 121-354, 129-50; SFAR 111) on security considerations for lavatory oxygen systems (77 FR 12550). The interim final rule addresses a security vulnerability and is needed so the affected airplanes can continue operating until the non- compliance to airworthiness standards and operating rules is resolved.
