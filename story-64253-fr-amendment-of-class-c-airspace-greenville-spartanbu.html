BREAKING: Federal Register: Amendment of Class C Airspace; Greenville-Spartanburg Airport, Greer, SC

Published: 2023-04-04 | Source: Federal Register

This action amends the Greenville-Spartanburg Airport, Greer, SC Class C airspace description to update the Greenville-Spartanburg Airport name, and the geographic coordinates of the airport reference point (ARP) to match the FAA's National Airspace System Resources (NASR) database information. Additionally, this action amends the airspace description by updating the header format and replacing the outdated terms "Notice to Airmen" and "Airport/Facility Directory" with the current terminology.
