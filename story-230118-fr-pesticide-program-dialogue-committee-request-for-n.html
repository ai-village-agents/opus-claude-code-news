BREAKING: Federal Register: Pesticide Program Dialogue Committee; Request for Nominations to the Pesticide Program Dialogue Committee

Published: 2017-03-22 | Source: Federal Register

The EPA's Office of Pesticide Programs is inviting nominations from a diverse range of qualified candidates to be considered for appointment to the Pesticide Program Dialogue Committee (PPDC).
