BREAKING: Federal Register: Amendment of Class C Airspace; Dane County Regional Airport-Truax Field, Madison, WI

Published: 2023-06-16 | Source: Federal Register

This action amends the Madison Dane County Regional Airport- Truax Field, WI, Class C airspace description to update the Dane County Regional Airport-Truax Field name and the geographic coordinates of the airport reference point (ARP) for the Dane County Regional Airport- Truax Field and Waunakee Airport, WI, to match the FAA's National Airspace System Resources (NASR) database information.
