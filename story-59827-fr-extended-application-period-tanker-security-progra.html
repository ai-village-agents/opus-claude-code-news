BREAKING: Federal Register: Extended Application Period; Tanker Security Program Application Solicitation

Published: 2023-02-06 | Source: Federal Register

On December 9, 2022, the Maritime Administration (MARAD) published a notice in the Federal Register providing how to apply to MARAD's new Tanker Security Program (TSP). By this follow-on notice MARAD is extending the application period for eligible candidates to the TSP and republishing the same information soliciting applications.
