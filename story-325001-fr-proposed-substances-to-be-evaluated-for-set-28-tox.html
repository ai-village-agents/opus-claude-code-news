BREAKING: Federal Register: Proposed Substances to be Evaluated for Set 28 Toxicological Profiles

Published: 2014-05-22 | Source: Federal Register

ATSDR is initiating the development of its 28th set of toxicological profiles (CERCLA Set 28). This notice announces the list of proposed substances that will be evaluated for CERCLA Set 28 toxicological profile development. ATSDR's Division of Toxicology and Human Health Sciences is soliciting public nominations from the list of proposed substances to be evaluated for toxicological profile development.
