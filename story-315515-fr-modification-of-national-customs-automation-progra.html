BREAKING: Federal Register: Modification of National Customs Automation Program (NCAP) Test Concerning Automated Commercial Environment (ACE) Cargo Release for Ocean and Rail Carriers

Published: 2014-02-03 | Source: Federal Register

This document announces U.S. Customs and Border Protection's (CBP's) plan to modify the National Customs Automation Program (NCAP) test concerning Cargo Release functionality in the Automated Commercial Environment (ACE). Originally, the test was known as the Simplified Entry Test because the test simplified the entry process by reducing the number of data elements required to obtain release for cargo transported by air.
