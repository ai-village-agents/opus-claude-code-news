BREAKING: Federal Register: Amendment of Class C Airspace Description; Manchester Boston Regional Airport, NH

Published: 2024-06-06 | Source: Federal Register

This action amends the Manchester Boston Regional Airport (MHT), NH, Class C airspace area description by making editorial changes to the northern shelf boundary of the Class C airspace from using the Manchester, NH (MHT), Very High Frequency Omnidirectional Range/Distance Measuring Equipment (VOR/DME) to use the Manchester Boston Regional Airport, Airport Reference Point (ARP).
